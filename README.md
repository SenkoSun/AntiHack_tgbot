# 🍸 AntiHack_tgbot

Telegram bot for the **AntiHack** hackathon.  
Works as a client for the "Black Barman" API — orders drinks, mixes cocktails, tracks balance and history.

## 🧠 Key features

- 📋 Menu with prices and ingredients
- 🍹 Order a drink by name
- 🔀 Create custom cocktails from ingredients
- 💰 Check balance
- 💸 Leave a tip
- 📜 Order history
- 🎴 User profile with rank and stats
- 🔁 Reset account

## 🎭 Personality

The barman has a **character** — sarcastic, grumpy, and slightly rude.  
He comments on every action based on your balance, rank, and behavior.

## 🛠 Tech stack

- Python 3.10+
- aiogram 3.x
- aiohttp
